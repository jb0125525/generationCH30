Términos random:

función
Es una sección de un programa que calcula un valor de manera independiente al resto del programa.
const
Nos permite declarar variables inmutables, osea no se puede cambiar.
var
Nos permiten crear variables mutables que si se puede cambiar.
figure
Establece una figura, que puede contener una serie de elementos diversos.
script
Se utiliza para utilizar Javascript en una página, debemos indicar en nuestro HTML que vamos a cargar un script (un archivo de texto con código Javascript) y hacerlo funcionar sobre el documento web actual.
details
Crea un elemento desplegable. Etiqueta contenedora.
